一、方案标题

家用简约宠物喂食器

二、方案应用场景

地点：家用住宅内；

功能：多种模式实现宠物喂食，逗自家宠物。1、不在家模式，可以通过手机网络实现宠物喂食。2、在家模式，分感应投食、撞击投食、手机网络投食。
